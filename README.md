# PDF Question Answering RAG using Gemini 2.0 Flash

This project is an end-to-end Retrieval-Augmented Generation (RAG) system built using LangChain and Google Gemini 2.0 Flash.

## Features
- Load and chunk PDFs
- Create embeddings and FAISS vector store
- Retrieve relevant context and generate answers
- Built using LangChain + Gemini 2.0 Flash
